# vyplayer
music player for your different vibes

A Three.js music player with immersive 3D scenes that react to audio.

## Stack
- React + Vite
- React Three Fiber (Three.js)
- Zustand (state)
- Web Audio API (audio analysis)
- iTunes Search API (music search + 30s previews)

## POC: 90s Winamp Vibe
- Spinning CD with rainbow grooves reacting to beat
- 16-bar audio-reactive VU meter
- Pixel star field + CRT scanline overlay
- 3 draggable/minimizable Winamp-style windows
- Search, playlist, playback controls

## Run
```bash
npm install
npm run dev
```
